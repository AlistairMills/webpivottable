WebPivotTable

WebPivotTable is a pivot table and pivot chart component which mimics all functionalities of Microsoft Excel Pivot Table with more intuitive user interface. It is pure javascript based so that it can be run totally in browsers and can be easily integrated into any web applications or any web pages.

Why WebPivotTable?

Pivot table is a very useful tool for data analyst. Microsoft Excel pivot table is the most popular implementation. Everyday, millions of users use it to analyze their data, create reports. It is also the major client side tool within Microsoft Business Intelligence solution to pivot Microsoft Analysis Service (SSAS) cubes interactively.

Despite the fact that pivot tables have been a cornerstone of Excel, they remain one of the most underutilized tools in the entire Microsoft Office Suite since they are not intuitive enough to be used by normal users and there are hard to be integrated into other applications by professional developers.

WebPivotTable comes to the rescue!

WebPivotTable is a pure javascript implementation of pivot table and pivot chart. Since the only requirement to run WebPivotTable is a browser, WebPivotTable can be easily used on any kind of devices (desktop computers, laptops, notebooks, tablets, smart phones, terminals, etc. ). Users can use the same WebPivotTable to analyst their data anytime anywhere on any devices.

WebPivotTable mimics all functionalities of Excel pivot table. By leverage the powerful web technology, it provides a very intuitive user interface for normal users. Most of WebPivotTable operations are just a simple drag and drop or a simple mouse click. There are almost no learn curves for normal users. Users can create a grid report or a chart report in a minute with tons of customize options. WebPivotTable is a great choice for any users to analysis their data easily and powerfully.

WebPivotTable supports to load data from all kinds of resources: a data file locate in local drive, a data file locate in a cloud drive or an Internet data link. The data format can be any of the most popular formats, like CSV, XLS, XLSX and Google spreadsheet. WebPivotTable also provide APIs (Application Programming Interface) for professional developers to load data from any back end SQL databases dynamically. It also support to pivot data directly on any OLAP (On Line Analysis Process) servers, like Microsoft Analyst Service, Pentaho Mondrian and icCube. All these supports makes WebPivotTable a great choice for professional developers to integrate it into their own applications or solutions.

A business intelligence solution

A lightweight business intelligence solution to analyze enterprise data and to share reports anytime anywhere on any devices.

  • A lightweight business intelligence solution

    WPT is a complete BI(Business Intelligence) solution. Users can load any source data into WPT, clean and refine the data, then pivot and analyze the data to create multiple dynamic pivot tables and pivot charts. Users can also combine these pivot tables and pivot charts to create any kind of reports and share all these analytics and reports with others.

    Not like tranditional BI solutions, WPT is lightweight. By leverage modern web 2.0 technologies, WPT put all BI operations into an unify intuitive web user interface. The very easy installation and setup process make it particularly suit for medium and small businesses to create a simple but powerful data analytics and reporting system.

  • To analyze enterprise data

    WPT provides dynamically and intuitively data analytics to everybody within enterprise. Enterprise managers, especially executive managers, need not waiting for weekly or monthly static reports to make their decisions any more. They can do all kinds of multiple dimensional analysis right away by themselves anytime with intuitive drag and drops.

  • And to share reports

    WPT also provides a reporting system to everybody within enterprise. Users can save their analytics results as reports anytime. Reports can be saved as files or be saved into database and be opened by web links anytime later on.

    Sharing reports to others can be achieved within system and any authorized report receiver can reopen the report by follow a web link to do further analysis. Users can also make comments to any report so as to communicate with others.

  • Anytime anywhere on any devices

    WPT is pure web based. The only requirement to use WPT is a modern internet browser. All user interfaces are based on latest HTML5 and javascript technology and can be accessed by users any time anywhere on any devices.

A javascript component

A pure javascript pivot table and pivot chart component which can be integrated into any web application or web site.

  • A pure javascript pivot Table and pivot chart component

    The core of WPT is a front end javascript component. This component looks like a web version Microsoft Excel Pivot Table. It provides almost all Excel Pivot Table functionalities in browsers with more intuitive user interfaces.

  • Which can be integrated into any web application or web site

    Since the component is a pure javascript component, it can be easily be integrated into any web site or any web application. There are a lots of APIs for integrations developers to use.

Powerful

Powerful is the key!

  • Widely data sources support

    • Data file as data source

      WPT supports to load and analyze all kinds of enterprise data. It provides an unify user interface for users to load their source data from data files into system, no matter it is in CSV format, Microsoft Excel format, or Google Spreadsheet format, and no matter it locates on users local drive, or cloud drives, or any web server as an internet link.

      tools for user to clean and reorganize the source data.

    • OLAP Cube as data source

      WPT support access Online analytics process (OLAP) cubes directly through XML for Analysis (XMLA) protocol. XMLA is a Simple Object Access Protocol (SOAP)-based XML protocol, designed specifically for universal data access to any standard multidimensional data source residing on the Web.

      Almost all major BI providers support this industry standard so that WPT can easily access their OLAP engines, like Microsoft Analytics Service, Mondrian open source OLAP server, Oracle Essbase, IBM DB2 OLAP server, SAS OLAP server, SAP OLAP server, etc.

      This is a great news for big companies who invested on creating OLAP cubes already. WPT can help these companies to easily expand the use of OLAP cubes to a wider community of users, especially enterprise executive managers.

    • SQL database and big data database as data source

      WPT provides a lots of APIs to load source data from backend database system. This includes all kinds of SQL relational databases and big data databases.

  • Full functional data analytics

    • Creating Excel-like pivot table and pivot chart in minutes

      The pivot table is a very useful tool for the data analyst. There are a lot of pivot table implementations in the market. Among all the implementations, Microsoft Excel is definitely the best one. Everyday, millions of users use it to analyze their data, to create their reports.

      WPT looks like a web version Excel. It implements almost all functionalities of Excel Pivot Table but with more intuitive user interface. What you can do with Excel pivot table, you can do the same in WPT.

    • Creating multiple pivot tables at same time

      WPT supports to create multiple pivot tables and pivot charts at same time. Each pivot tables and charts can be customized independently and they can based on different data sources.

    • Easily pivot operations with cutomized summarize functions and data formats

      WPT supports customize all spects of pivot operation. Changing summarize functions, changing sorts and filters, changes data formats are very simple and easy. And user can see these changes immidiately.

    • Powerful cutomized options for pivot table and pivot chart

      WPT supports to customize almost all aspects of pivot table and pivot chart. For example: table size, table theme, table layout, totals and subtotals, chart size, chart types, chart themes, chart legends, tooltips, etc. All these changes are very intuitive and user can see these changes immidiately.

How to use WPT?

WPT can be used as an independent enterprise BI solution, it can be also integrated into any other web application or web site. OEM or SaaS integrations are also supported.

Use as an independent enterprise business intelligence solution

  • Install open source Node.js platform on your server machine. Run "node --version" to check whether Node.js has been installed correctly.
  • Install open source MongoDB NoSQL database on the same server machine or another server machine in the same network. After MongoDB installation, start MongoDB follow the MongoDB document If you don't want install MongoDB, you can try use cloud based MongoDB. There are lots of cloud based MongoDB in the market, for example, Compose (original MongoHQ) and MongoLab.
  • Download WPT and unzip it to a directory. If MongoDB is not on the same server machine, open "app/config.js" file to change "db" parameter to point to correct MongoDB URL.
  • Go to unzipped directory, run "npm install"
  • run "node app"
  • Open any browser, access http://localhost:8002

Integrate into any web application as javascript component

  • Download WPT and unzip it to a directory.
  • Copy all contents under public subdirectory into your web application www root directory.
